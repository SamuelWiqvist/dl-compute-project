This data set is from a study where a population was tested for
diabetes. The inputs contain information about the individuals and
the output is either a one, if they tested positive, or a zero, if
they tested negative. The data set contains 768 data points where
1/3 is reserved for the test set.

One can read quit a lot of it here:
https://www.andreagrandi.it/2018/04/14/machine-learning-pima-indians-diabetes/

The files are tab-separated. First column is case-ID and the last
column is the target. Column 2-9 contain the input variables.
